A font used for writing on a Layer.

Fonts can be created from files, registered system fonts or existing font objects. When creating fonts from system fonts, a FontInfo structure can be used to define requirements for the font, e.g. "any available monospace font".

In addition to the usual properties of a font, radiant also assigns a fixed color and size to each font object. Instead of modifying these properties, you can clone a new font with modified values using Font::with_color() and/or Font::with_size().
